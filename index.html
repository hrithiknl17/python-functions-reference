🐍 Python Functions Reference Guide

Complete reference for Python built-in functions and math module functions

Built-in Functions Math Functions Usage Examples

🔧 Python Built-in Functions

abs()
Returns the absolute value of a number
abs(x)
abs(-5) → 5
abs(3.14) → 3.14
all()
Returns True if all elements in an iterable are true
all(iterable)
all([True, True, True]) → True
all([True, False, True]) → False
any()
Returns True if any element in an iterable is true
any(iterable)
any([False, True, False]) → True
any([False, False, False]) → False
bin()
Converts an integer to a binary string
bin(x)
bin(10) → '0b1010'
bin(255) → '0b11111111'
bool()
Returns a Boolean value
bool([x])
bool(1) → True
bool(0) → False
bool([]) → False
chr()
Returns a character from the given Unicode code point
chr(i)
chr(65) → 'A'
chr(97) → 'a'
dict()
Creates a dictionary
dict(**kwarg)
dict(mapping, **kwarg)
dict(iterable, **kwarg)
dict(a=1, b=2) → {'a': 1, 'b': 2}
dict([('a', 1), ('b', 2)]) → {'a': 1, 'b': 2}
enumerate()
Returns an enumerate object
enumerate(iterable, start=0)
list(enumerate(['a', 'b', 'c'])) → [(0, 'a'), (1, 'b'), (2, 'c')]
eval()
Evaluates and executes an expression
eval(expression, globals, locals)
eval('2 + 3') → 5
eval('len("hello")') → 5
filter()
Constructs an iterator from elements which are true
filter(function, iterable)
list(filter(lambda x: x > 0, [-1, 0, 1, 2])) → [1, 2]
float()
Returns a floating point number
float([x])
float('3.14') → 3.14
float(5) → 5.0
format()
Returns a formatted representation of a value
format(value, format_spec)
format(3.14159, '.2f') → '3.14'
format(42, 'b') → '101010'
hex()
Converts an integer to a hexadecimal string
hex(x)
hex(255) → '0xff'
hex(16) → '0x10'
input()
Reads a line from input
input([prompt])
name = input("Enter name: ")
# User types "Alice"
name → "Alice"
int()
Returns an integer object
int([x])
int(x, base=10)
int('42') → 42
int('1010', 2) → 10
len()
Returns the length of an object
len(s)
len("hello") → 5
len([1, 2, 3]) → 3
list()
Creates a list
list([iterable])
list("hello") → ['h', 'e', 'l', 'l', 'o']
list(range(3)) → [0, 1, 2]
map()
Returns an iterator that applies function to every item
map(function, iterable, ...)
list(map(lambda x: x**2, [1, 2, 3])) → [1, 4, 9]
max()
Returns the largest item
max(iterable, *[, key, default])
max(arg1, arg2, *args[, key])
max([1, 2, 3]) → 3
max('hello') → 'o'
min()
Returns the smallest item
min(iterable, *[, key, default])
min(arg1, arg2, *args[, key])
min([1, 2, 3]) → 1
min('hello') → 'e'
oct()
Converts an integer to an octal string
oct(x)
oct(8) → '0o10'
oct(64) → '0o100'
ord()
Returns the Unicode code point of a character
ord(c)
ord('A') → 65
ord('a') → 97
pow()
Returns x to the power y
pow(base, exp[, mod])
pow(2, 3) → 8
pow(2, 3, 3) → 2
print()
Prints objects to the text stream
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
print("Hello", "World") → Hello World
print(1, 2, 3, sep='-') → 1-2-3
range()
Returns a sequence of numbers
range(stop)
range(start, stop[, step])
list(range(5)) → [0, 1, 2, 3, 4]
list(range(1, 6, 2)) → [1, 3, 5]
reversed()
Returns a reverse iterator
reversed(seq)
list(reversed([1, 2, 3])) → [3, 2, 1]
list(reversed("hello")) → ['o', 'l', 'l', 'e', 'h']
round()
Returns a number rounded to ndigits precision
round(number[, ndigits])
round(3.14159, 2) → 3.14
round(1234.5, -1) → 1230.0
set()
Creates a set object
set([iterable])
set([1, 2, 2, 3]) → {1, 2, 3}
set("hello") → {'h', 'e', 'l', 'o'}
sorted()
Returns a new sorted list
sorted(iterable, *, key=None, reverse=False)
sorted([3, 1, 2]) → [1, 2, 3]
sorted("hello") → ['e', 'h', 'l', 'l', 'o']
str()
Returns a string object
str(object='')
str(object=b'', encoding='utf-8', errors='strict')
str(42) → '42'
str([1, 2, 3]) → '[1, 2, 3]'
sum()
Sums start and the items of an iterable
sum(iterable, /, start=0)
sum([1, 2, 3, 4]) → 10
sum([1, 2, 3], 10) → 16
tuple()
Creates a tuple
tuple([iterable])
tuple([1, 2, 3]) → (1, 2, 3)
tuple("hello") → ('h', 'e', 'l', 'l', 'o')
type()
Returns the type of an object
type(object)
type(42) → <class 'int'>
type("hello") → <class 'str'>
zip()
Returns an iterator of tuples
zip(*iterables)
list(zip([1, 2], ['a', 'b'])) → [(1, 'a'), (2, 'b')]

🔢 Python Math Module Functions

math.ceil()
Returns the ceiling of x
import math
math.ceil(x)
math.ceil(4.2) → 5
math.ceil(-3.8) → -3
math.floor()
Returns the floor of x
import math
math.floor(x)
math.floor(4.8) → 4
math.floor(-3.2) → -4
math.sqrt()
Returns the square root of x
import math
math.sqrt(x)
math.sqrt(16) → 4.0
math.sqrt(2) → 1.414...
math.pow()
Returns x raised to the power y
import math
math.pow(x, y)
math.pow(2, 3) → 8.0
math.pow(9, 0.5) → 3.0
math.log()
Returns the natural logarithm of x
import math
math.log(x[, base])
math.log(math.e) → 1.0
math.log(8, 2) → 3.0
math.log10()
Returns the base-10 logarithm of x
import math
math.log10(x)
math.log10(100) → 2.0
math.log10(1000) → 3.0
math.sin()
Returns the sine of x (in radians)
import math
math.sin(x)
math.sin(math.pi/2) → 1.0
math.sin(0) → 0.0
math.cos()
Returns the cosine of x (in radians)
import math
math.cos(x)
math.cos(0) → 1.0
math.cos(math.pi) → -1.0
math.tan()
Returns the tangent of x (in radians)
import math
math.tan(x)
math.tan(0) → 0.0
math.tan(math.pi/4) → 1.0
math.asin()
Returns the arc sine of x in radians
import math
math.asin(x)
math.asin(1) → 1.570...
math.asin(0.5) → 0.523...
math.acos()
Returns the arc cosine of x in radians
import math
math.acos(x)
math.acos(1) → 0.0
math.acos(0) → 1.570...
math.atan()
Returns the arc tangent of x in radians
import math
math.atan(x)
math.atan(1) → 0.785...
math.atan(0) → 0.0
math.degrees()
Converts angle x from radians to degrees
import math
math.degrees(x)
math.degrees(math.pi) → 180.0
math.degrees(math.pi/2) → 90.0
math.radians()
Converts angle x from degrees to radians
import math
math.radians(x)
math.radians(180) → 3.141...
math.radians(90) → 1.570...
math.exp()
Returns e raised to the power x
import math
math.exp(x)
math.exp(0) → 1.0
math.exp(1) → 2.718...
math.fabs()
Returns the absolute value of x
import math
math.fabs(x)
math.fabs(-5.5) → 5.5
math.fabs(3.2) → 3.2
math.factorial()
Returns x factorial
import math
math.factorial(x)
math.factorial(5) → 120
math.factorial(0) → 1
math.gcd()
Returns the greatest common divisor
import math
math.gcd(a, b)
math.gcd(12, 8) → 4
math.gcd(15, 25) → 5
math.isnan()
Check if x is NaN (not a number)
import math
math.isnan(x)
math.isnan(float('nan')) → True
math.isnan(5.0) → False
math.isinf()
Check if x is infinite
import math
math.isinf(x)
math.isinf(float('inf')) → True
math.isinf(100) → False
math.isfinite()
Check if x is finite
import math
math.isfinite(x)
math.isfinite(5.0) → True
math.isfinite(float('inf')) → False
math.trunc()
Truncates x to the nearest integer toward zero
import math
math.trunc(x)
math.trunc(3.7) → 3
math.trunc(-3.7) → -3

💡 Practical Usage Examples

Data Processing
Common data manipulation tasks
# Filter positive numbers
numbers = [-2, -1, 0, 1, 2, 3]
positive = list(filter(lambda x: x > 0, numbers))
positive → [1, 2, 3]
Mathematical Calculations
Complex mathematical operations
# Calculate distance between points
import math
distance = math.sqrt((x2-x1)**2 + (y2-y1)**2)
For points (0,0) and (3,4): distance = 5.0
Number Systems
Converting between different number bases
# Convert decimal to different bases
num = 255
binary = bin(num) # '0b11111111'
octal = oct(num) # '0o377'
hexadecimal = hex(num) # '0xff'
255 → Binary: 11111111, Octal: 377, Hex: ff

🎯 Quick Reference Tips

  • 🔸 Import math module: Always use import math before using math functions
  • 🔸 Angles in radians: Trigonometric functions use radians, not degrees
  • 🔸 Type conversion: Use int(), float(), str() for type conversion
  • 🔸 Iterables: Most functions work with lists, tuples, strings, and other iterables
  • 🔸 Performance: Built-in functions are optimized and usually faster than custom implementations
